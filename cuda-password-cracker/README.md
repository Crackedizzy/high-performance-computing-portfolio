# ⚡ CUDA Password Cracker

## 🧠 Description
GPU-accelerated brute-force password cracker using CUDA.  
Searches the keyspace using a custom encryption function (`CudaCrypt`).

## 🛠️ How to Compile
```bash
make
```

## ▶️ How to Run
```bash
./password_crack_cuda
```

## 🧹 Clean Up
```bash
make clean
```
